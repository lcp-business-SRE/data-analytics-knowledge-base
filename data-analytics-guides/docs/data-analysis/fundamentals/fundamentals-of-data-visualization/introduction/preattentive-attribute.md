# 1-4. ヒトがグラフを一瞬で理解できる仕組みを知ろう
ヒトはグラフを見ると、一瞬で重要な情報を理解できます。  
これは、グラフを見たときに、色や形、配置などをもとに無意識に情報の優先順位をつけているからです。  

例えば、以下の図を見てください。

上も下も数字の並びは全く同じですが、下の図では5秒以内に「10個」あると数えることができたのではないでしょうか？  
このように、文字の色を変えただけで必要な情報を瞬時に把握することができます。  

今回で言う**色**や、**形**、**長さ**、**位置**、**向き**、**大きさ**などの視覚的な特徴のことを**視覚属性**と呼びます。  
**データの可視化は、この視覚属性を活用してヒトが一瞬で理解できる**ようにしているのです。

例えば、自社と競合他社の売上を比較するグラフを作成してみましょう。

棒グラフは、視覚属性の**長さ**を用いたグラフです。  
**棒の長さという視覚属性によって、簡単に量を比較することができる**のです。  
さらに、このグラフは自社だけ青色にすることで、自然と自社のデータに注目を集めています。  
このように、**伝えたいことに合わせて視覚属性を活用する**ことも可能です。
